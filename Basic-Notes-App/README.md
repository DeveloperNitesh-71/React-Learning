# React Notes App

A simple notes application built using **React fundamentals**.  
This project was created to practice core React concepts such as components, state management, and basic UI interactions.

---

## 🎯 Project Purpose

The goal of this project is to understand and apply fundamental React concepts including:

- Component-based architecture
- State management using React hooks
- Handling user interactions
- Dynamic UI rendering

---

## 🚀 Features

- Create new notes
- Display notes dynamically
- Simple and clean user interface
- Component-based structure
- Responsive layout

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6)
- HTML5
- CSS3

---

## 🧠 React Concepts Used

- Functional Components
- React Hooks (useState)
- Props
- Event Handling
- Conditional Rendering
